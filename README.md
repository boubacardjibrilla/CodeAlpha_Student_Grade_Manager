# CodeAlpha Student Grade Manager

A Java Swing desktop application to manage students and their grades.

## Features
- Create student profiles
- Add grades
- Calculate average, highest and lowest grades
- GUI interface using Java Swing

## Tech Stack
- Java
- Swing (JFrame)
- OOP concepts
